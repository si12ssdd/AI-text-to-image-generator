# AI Text-to-Image Generator 🎨🤖

A full-stack AI-powered web application that converts text prompts into high-quality images using modern frontend and backend technologies.

---

## 🚀 Live Demo
**Frontend:** https://text-to-image-frontend-sigma.vercel.app/
**Backend API:** https://text-to-image-backend-plum.vercel.app/

---

## 🛠️ Tech Stack
- **Frontend:** React (Vite), Tailwind CSS
- **Backend:** Node.js, Express
- **AI Integration:** Text-to-Image API
- **Deployment:** Vercel (Client & Server)

---

## ✨ Features
- Text-to-image generation using AI
- User authentication (login/signup)
- Credit-based image generation
- Responsive UI
- Production-ready deployment

---

## 📂 Project Structure
├── client # Frontend (React + Vite)
└── server # Backend (Node.js + Express)



---

## 🧑‍💻 How It Works
1. User enters a text prompt on the frontend
2. Frontend sends request to backend API
3. Backend processes request and returns generated image
4. Image is displayed instantly on UI

---

## 📌 Note
The backend is API-based, so visiting the root URL may return 404 unless a specific API route is accessed.
